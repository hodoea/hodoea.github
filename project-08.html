Startupmatcher Re-design sketch

This is a re-design of startupmatcher first front-page where I currently work as an interaction designer. The original website did not have any galleries on their front-page and it consisted mostly of the welcome title and a button. My suggestion was verticle galleries to display the jobs immediately in order to trigger the user to create a profile. They also have a database of people's profile which is searchable for everyone. You can see the whole website here: www.startupmatcher.com.
